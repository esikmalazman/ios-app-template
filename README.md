# iOS App Template
This repository consists of a template for building a SwiftUI iOS app in Xcode.

## Project Structure

- `example-project/` — App source code, SwiftUI views, assets
- `example-project.xcodeproj/` — Xcode project files
- `example-project/instructions/instructions.mdc` — App requirements document
- `.cursor/rules/` — Cursor project rules
- `AGENTS.md` — Agent guidance for this repository

## Source of Truth

- Product/app requirements: `example-project/instructions/instructions.mdc`
- AI behavior and coding guidance: `AGENTS.md` and `.cursor/rules/*.mdc`
